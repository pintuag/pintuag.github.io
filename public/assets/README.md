# Project Images

Add your project images here with the following names:

1. **project-spreeh.png** - Spreeh wedding and event management platform screenshot
2. **project-rupeek.png** - Rupeek gold loan disbursement platform screenshot  
3. **project-modi11.png** - Modi 11 mobile content platform screenshot

## Recommended Image Specifications

- **Format**: PNG or JPG
- **Dimensions**: 800x600 pixels (or 16:9 / 4:3 aspect ratio)
- **File size**: Optimize images to be under 500KB for fast loading

## How it works

Once you add images here, they will be served at:
- `/assets/project-spreeh.png`
- `/assets/project-rupeek.png`
- `/assets/project-modi11.png`

These paths are already configured in `src/lib/placeholder-images.json`.
