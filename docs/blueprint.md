# **App Name**: ArcFlow Architect

## Core Features:

- Dynamic Content Loading: Load all portfolio content (About, Experience, Projects, Skills, Platform Engineering, AI Engineering) from local structured JSON/Markdown files, allowing easy updates without redeploying code.
- Interactive Hero Section & Navigation: Present a high-impact hero section with name, title, intro, animated stats, social links, and a sticky top navigation bar with smooth scroll to content sections.
- Experience Timeline Showcase: Display work experience in an interactive, vertically aligned timeline, where each entry expands to reveal detailed responsibilities and achievements.
- Modern Project Grid: Showcase engineering projects using a card-based grid layout with animated interactions, linking to detailed project information.
- Categorized Skills Overview: Visualize technical proficiencies through a categorized display (e.g., chips or cards for Languages, Backend, Mobile, etc.) highlighting expertise.
- AI-Augmented Insight Tool: Utilize an AI model as a tool to generate succinct, engaging summaries or highlight key innovations from provided project descriptions or experience entries, reflecting AI-augmented development.
- Responsive & Themeable UI: Implement a fully responsive design for all devices, including a dark mode enabled by default with an optional light mode toggle, ensuring a consistent premium user experience.

## Style Guidelines:

- Primary color: '#4F46E5' (Indigo), reflecting professionalism and modern tech.
- Secondary color: '#6366F1' (Lighter Indigo), used for softer highlights and accents that complement the primary.
- Accent color: '#22C55E' (Emerald Green), a vibrant contrasting color for interactive elements and call-to-actions.
- Dark Background color: '#0B0F19' (Very Dark Blue-Gray), providing a deep, high-contrast canvas for the interface elements.
- Surface color: '#111827' (Darker Blue-Gray), used for card backgrounds and other UI containers within the dark theme.
- Primary text color: '#E5E7EB' (Off-White), ensuring high readability on dark backgrounds.
- Secondary text color: '#9CA3AF' (Gray), for supplementary information and less prominent text elements.
- Primary font: 'Inter' (sans-serif) for general text and headings, offering a modern, neutral, and highly readable aesthetic. Code font: 'Roboto Mono' (monospace) for displaying code snippets, ensuring clear and structured code presentation. Note: currently only Google Fonts are supported.
- Utilize modern, clean line icons from either 'Lucide Icons' or 'Material Icons' to maintain a crisp and professional visual language throughout the portfolio.
- Implement a minimalist design with ample whitespace, a sticky top navigation bar, card-based content sections, and an interactive vertical timeline for experience. Subtle gradients and glass effects will enhance the visual appeal, alongside a fully responsive structure for all devices.
- Integrate smooth Framer Motion-driven animations for section transitions, animated statistics cards in the hero, dynamic project cards, and interactive elements within the experience timeline. A subtle animated background gradient or grid pattern will add a high-end feel.
